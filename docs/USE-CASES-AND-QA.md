## A. Storefront chat — Shopping (Concierge)
**Does:** greets, captures lead, recommends coffee via the deterministic strategy engine (never invents products/prices).
**QA:** open chat → "I want a smooth medium roast for espresso" → agent asks/uses prefs, recommends real products + quantities.
**Where:** chat widget; behind the scenes `RecommendationStrategyService`. Check `sf data query "SELECT Name FROM Product2 WHERE IsActive=true"` matches what it recommends.
**Pass:** recommends only real catalog products; no invented prices/discounts.
